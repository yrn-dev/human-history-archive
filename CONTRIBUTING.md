# Contributing to Human History Archive

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Add your historical content
4. Submit a pull request

## Content Guidelines

- Use markdown format (`.md`)
- Include dates and sources
- Write in English (other languages welcome)
- Keep articles concise but informative
- Cite sources where possible

## File Structure

```
eras/<era>/<period>.md
events/<category>/<event>.md
cultures/<culture>/<topic>.md
```

## AI-Assisted Content

This archive uses AI assistance for content generation. All content is reviewed for historical accuracy.
